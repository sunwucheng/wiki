2017_声发射信号处理算法研究

来源：声发射信号处理算法研究

1 绪论

1.2 无损检测技术分析

无损检测技术：目视法、渗透法、超声波法、X射线法、计算机层析照相法(CT)、热成像法、微波法、声发射检测法

1.3 声发射检测技术研究现状及分析

1.3.1 声发射检测技术概念

声发射(Acoustic Emission, AE)/应力波发射(Stress Wave Emission)是指材料内部或表面由于变形或者损坏而突然释放应变能产生的瞬态弹性波

声发射源 含义 例子
一次声发射源 直接由于变形和断裂而产生的弹性波源 金属塑性变形、断裂、相变、磁效应、表面效应
二次声发射源 与变形和断裂没有直接关系的声发射源 流体泄漏、摩擦、撞击、燃烧

1.3.2 声发射信号采集处理系统

三个阶段 声发射信号采集处理系统的硬件系统 代表产品
1965-1983 模拟式声发射仪器时代 （已基本淘汰）

1995- 全数字式声发射仪器时代 德国VALLEN的AMSY-6

1.3.3 声发射信号分析技术

声发射检测检测方法 描述
参数分析法 利用简化的波形特征来描述声发射信号特征
波形分析法 对声发射信号波形进行分析获取声发射信号特征

1.3.3.1 参数分析法

常用的声发射参数：幅度、能量、上升时间、持续时间、振铃技术、事件技术
基于参数分析法的检测标准：美国ASME标准、国家标准GB/T 18182
参数分析法假设声发射信号在介质中的传播速度是恒定的，其实际传播速度为

$v_{1}=\sqrt{\frac{E(1-\sigma)}{\rho(1+\sigma)(1-2 \sigma)}}，v_{2}=\sqrt{\frac{E}{\rho(2+2 \sigma)}}=\sqrt{\frac{G}{\rho}}$

$v_{1}：纵波速度；v_{2}：横波速度；σ：介质的泊松比；E：介质的杨氏弹性模量；G：介质的切变模量；ρ：介质的密度$

1.3.3.2 波形分析法

波形分析法 通过对记录和存储的声发射信号进行时域分析、频域分析和时频域分析来获取声发射源特性
频谱分析法 通过对声发射信号进行傅里叶变换，获取其频谱特性，提取频域中的声发射信号特征
模态分析法 基于波板理论，旨在分析声发射地物理机制，主要考虑两种0阶板波模式，扩展波和弯曲波
时频域分析 时-频分析方法克服了基于傅里叶变换的频域分析法的不足，能够同时提取信号的时域和频域特性

频谱分析只考虑了信号的全局信息而忽略了信号的局部信息，对于周期平稳信号进行分析是非常有效的，而对于声发射信号这种非平稳随机信号，对其进行频谱分析只是获取了信号的趋势结果，不能有效地提取声发射信号地局部非平稳特征。

模态声发射分析方法在源识别精度和源定位精度上都有很大的优势，但需能正确识别E波和F波，且试件的集合形状及声发射波在试件上的衰减特性都会大大影响模态声发射的处理精度。
扩展波：简称E波，是位移与板平面平行的板波
弯曲波：简称F波，是位移与板平面垂直的板波

目前应用比较广泛的时-频分析方法主要有短时傅里叶变换、Wigner-Ville分布、小波变换

  • 短时傅里叶变换：改进传统傅里叶变换，用长度和形状固定的窗函数实现局部分析
    由于采用固定的窗函数，短时傅里叶变换不能兼顾频率与时间分辨率的需求，适合于分析分段平稳信号或近似平稳信号
  • Wigner-Ville分布：相较短时傅里叶变换时频分辨率有很大提高，但存在交叉干扰
  • 小波分析：基于多分辨率框架，具有良好的时-频局域性，特别适合非平稳信号的分析，能提取更丰富的声发射源信号
    小波分析主要用于去噪、特征提取、声发射源定位三方面

1.3.4 声发射信号去噪技术

声发射信号处理困难：（1）声发射信号本身很微弱；（2）声发射信号易受噪声干扰
信号去噪的主要方法：时域分析法、频域分析法、时-频分析方法（主要基于小波变换）

  • 基于小波分析的去噪方法很适合对复合材料部件损伤产生的突发型非平稳声发射信号处理
    小波去噪缺点：去噪效果、信号特点与小波基函数关系很大，信噪比较小时去噪效果不好
  • 经验模态分解(Empirical mode decomposition, EMD)将信号分解为很多个本征模太分量(Intrinsic mode function, IMF)和一个余项之和
  • 独立分量分析(Independent Component Analysis, ICA)从盲源分离发展而来

1.3.5 声发射源定位技术

声发射源定位方法 描述 举例
参数定位法 根据各种声发射参数来识别声发射源位置 时差定位法、能量定位法、区域定位法、智能定位法
波形定位法 主要基于板波理论的模态分析定位法

1.3.6 声发射信号特征参数提取技术

特征提取：通过映射变换从原始数据中提取得到特征向量
特征选择：从已有的特征中挑选最有效的特征实现特征降维
通过使用相应的信号分析和数据处理方法，把原始声发射信号转换为能描述声发射源位置或状态的特征量，然后通过特征选择方法把无关的特征信息量去掉

基于传统的参数分析法，可提取的声发射信号特征参数有：信号能量、信号幅值、绝对能量、信号上升时间、信号持续时间、振铃技术、峰值计数、平均频率、初始频率、有效值电压、平均信号电压值、信号强度、频域质心、峰值频率等。
特征选择方法：投影寻踪法，主成分分析法、独立成分分析和相应的核方法，神经网络和遗传算法
主元分析(PCA)是统计学中非常有效的数据降维方法，PCA提取出能代表原始信息的最少数目的主分量，并保留原始变量的最多信息
PCA本质上是线性变换，只能有效的从线性相关的特征参数中进行选择
声发射信号参数间存在非线性，当使用PCA方法进行特征选择时，将选择非最有效的甚至错误的特征参数
为解决该问题，具有非线性处理能力的核主成分分析(KPCA)被用于声发射信号的特征提取

1.3.7 声发射源识别技术

声发射源类型识别，如损伤类型的识别本质上是模式识别的问题
模式识别方法主要包括三大类：决策理论方法/统计模式识别方法、句法方法、模糊判别方法

用于声发射源识别的模式识别方法主要是基于神经网络，k-近邻分类法，都是基于统计模式识别方法
一种在小样本情况下具有良好推广性的机器学习方法：支持向量机(SVM)
支持向量机以统计学习理论(Statistical Learning Theory, SLT)为基础，基于结构风险最小化(Structural Risk Minimization)，具有良好的小样本和非线性模式识别性能

2 声发射信号传播特性实验研究

2.1 引言

传统的小波包分析存在很多缺点，使其在分析精度、使用灵活性上存在一定的局限性

  • 不具有理想盒形的频谱特性，造成相邻频段能量泄露
  • 频带以2的倍数划分，不能任意选取感兴趣的频带进行分析
  • 每分解一层小波系数就会减半，造成不同层频率分辨率不同

谐波小波具有理想盒形的频谱特性，谐波小波包分析克服了以上缺点

2.3 声发射信号传播特性分析

测试信号一般由标准断铅试验产生，即Nielson源测试法，该方法利用与被测试件成一定角度的0.5mm的2H铅笔芯瞬间断裂产生的信号模拟试件裂纹所产生的声发射信号

5 声发射信号识别算法研究

5.1 引言

声发射信号频率成分非常丰富，是典型的非平稳信号，基于单纯时域或频域的特征提取方法都不能有效提取声发射信号特征。
基于时-频分析方法的声发射信号特征提取主要是基于小波分析和小波包分析的特征提取方法。
传统的小波（包）分析存在能量泄露、频带选取不灵活、不同层频率分辨率不同的缺点，并且基函数选择也没有统一的标准。

5.2 基于谐波小波包分解的声发射信号特征提取

小波变换作为最经典的时-频分析方法，非常适合对非平稳信号或时变信号进行分析，但是小波变换存在高频低分辨率问题，即信号在进行小波分解时，只是对低频部分进行了进一步分解。为此，小波包应运而生，其具有更好的频域特性，对信号的高频和低频部分都进行了分解，大大提高了频率分辨率。但是小波包分析在分析精度、使用灵活性上存在一定局限性。为此，谐波小波克服上述缺点，具有理想盒形的频谱特性。

5.3 基于支持向量多分类的声发射源类型识别

支持向量机在有限样本情况下，对模型复杂性和学习能力寻求最佳折中，寻求最佳的泛化能力
支持向量机的原型是处理两分类问题的，而声发射源存在多种类型，即声发射源识别是多分类问题，因此需要研究基于支持向量机的多分类方法，目前主要有两类：直接求解法、将多分类问题转化为多个二分类问题进行求解。

  • 直接求解法通过建立基于多分类的目标函数进行求解，由于优化过程变量多，计算效率低
  • 将多分类问题转化为多个二分类问题进行求解的方法根据两分类机的确定和组织方法不同，可分为一对多法、一对一法、层次SVM多类分类法
